# RÚBRICA DE CONTENIDO Y EVALUACIÓN

## Distribución de pesos del documento escrito

| Sección | Puntos |
|---------|--------|
| Introducción | 1 punto |
| Análisis | 1.5 puntos |
| Valoración económica | 1 punto |
| Viabilidad del proyecto | 1 punto |
| Puesta en marcha | 2 puntos |
| Evaluación funcional | 1.5 puntos |
| Conclusión | 1 punto |
| Bibliografía | 1 punto |
| **TOTAL** | **10 puntos** |

## NIVEL ALTO (lo que hay que conseguir)

### 1. Introducción
✓ Contextualiza y justifica la elección del tema

### 2. Análisis (1.5 pts)
✓ Analiza entorno de la empresa
✓ Valora necesidades y demandas
✓ Estudio de mercado (soluciones propietarias y libres)
✓ Valoración con criterios de re-usabilidad, complejidad
✓ Filosofía KISS

### 3. Valoración económica (1 pt)
✓ Análisis completo de costes desarrollo Y mantenimiento
✓ Hardware, licencias, personal

### 4. Viabilidad del proyecto (1 pt)
✓ Análisis realista
✓ Riesgos
✓ Análisis económicos
✓ Beneficios objetivos
✓ Interés real

### 5. Puesta en marcha (2 pts)
✓ Trámites burocráticos
✓ RRHH
✓ Prevención de riesgos laborales
✓ Análisis medioambiental
✓ Necesidades de inversión
✓ Recursos propios y ajenos
✓ Ayudas y subvenciones

### 6. Evaluación funcional (1.5 pts)
✓ Funcionalidad de la solución
✓ Si se han logrado las metas propuestas
✓ Herramientas utilizadas para comprobarlo

### 7. Conclusión (1 pt)
✓ Ideas finales del proyecto
✓ Resumiendo o englobando el proyecto
✓ Preguntas para abrir debate

### 8. Bibliografía (1 pt)
✓ Referencias actualizadas
✓ Citas siguiendo formato APA 7

## RÚBRICA FORMATO

### Claridad de ideas (30 pts)
- Ideas organizadas
- Sintaxis correcta
- Signos de puntuación correctos

### Estructura y coherencia (20 pts)
- Títulos, encabezados que organizan
- Párrafos lógicos

### Vocabulario, gramática y ortografía (30 pts)
- Tecnicismos adecuados
- Tiempos verbales adecuados
- Sin errores de ortografía

### Formalidad (20 pts)
- Misma fuente en títulos/subtítulos/cuadros
- Signos adecuados
- Índice y páginas
- Margen e interlineado correctos
