# GUÍA PARA LA REDACCIÓN DE TFC - CURSO 2024/2025

## 1. Características generales
- Proyecto de **emprendimiento** (no mera exposición)
- Tema acordado entre profesor y estudiante

## 2. Cuestiones formales
| Aspecto | Valor |
|---------|-------|
| Tamaño página | DIN A4 |
| Extensión | 25-30 páginas |
| Margen superior | 2 cm |
| Margen inferior | 2 cm |
| Margen izquierdo | 2.5 cm |
| Margen derecho | 2.5 cm |
| Fuente cuerpo | Times New Roman 12 |
| Fuente pie | Times New Roman 10 |
| Interlineado cuerpo | 1.5 |
| Interlineado pie | Simple |

## 3. Estructura del TFC
1. Portada
2. Resumen/abstract (150-250 palabras) + 5 palabras clave (ES/EN, alfabético, separadas por ;)
3. Índice
4. Introducción
5. Análisis
6. Valoración económica
7. Viabilidad del proyecto
8. Puesta en marcha
9. Evaluación funcional
10. Conclusión
11. Bibliografía
12. Anexos

## 4. Citación
APA 7ª edición obligatoriamente

## 5. Plagio
Máximo 30% de coincidencias

## 6. Nota final

|----------|------|
| Documento escrito | 70% |
| Emprendimiento | 20% |
| Soporte exposición | 15% |
